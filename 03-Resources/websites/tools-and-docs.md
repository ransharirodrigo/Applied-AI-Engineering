# 🌐 Key Tools & Documentation

## 🛠️ Essential Engineering Tools
* **[OpenAI Tokenizer](https://platform.openai.com/tokenizer):** A visual tool to understand how text is broken into tokens and how to stay within context window limits.
* **[Tiktoken (GitHub)](https://github.com/openai/tiktoken):** The official Python library for counting tokens. Essential for calculating API costs in Phase 4.

## 📚 Official Documentation
* **[OpenAI Cookbook](https://cookbook.openai.com):** High-quality code examples for everything from prompt engineering to RAG and Fine-tuning.
* **[Anthropic Model Context Protocol (MCP)](https://modelcontextprotocol.io):** An open standard that enables LLMs to connect to data sources and tools (Key for Agentic workflows).
