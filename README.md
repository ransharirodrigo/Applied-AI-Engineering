## 🧠 What's Covered

### Phase 1 — LLM Foundations & Prompt Engineering
- How LLMs work: tokenization, context windows, next-token prediction
- Prompting techniques: Zero-shot, Few-shot, Chain-of-Thought (CoT), ReAct
- Prompt evaluation: BLEU/ROUGE scoring, OpenAI Cookbook

### Phase 2 — Python & Mathematics for AI
- Python for data: NumPy and Pandas essentials
- Mathematics: Linear Algebra (vectors/matrices), Probability & Statistics
- Classical ML: Linear Regression theory and implementation

### Phase 3 — Deep Learning & NLP
- Neural Networks: backpropagation, activation functions, loss functions
- NLP: text preprocessing, Word Embeddings (Word2Vec/GloVe), sentiment analysis
- Transformers: the attention mechanism explained
- Fine-tuning: LoRA and PEFT concepts

### Phase 4 — Advanced AI Orchestration
- RAG (Retrieval Augmented Generation): architecture and implementation
- Vector Databases: Pinecone and ChromaDB
- LangChain / LangGraph: orchestration fundamentals
- AI Agents: function calling, tool use, autonomous workflows
- Multi-Agent Systems: CrewAI basics
- Model Evaluation: accuracy, precision, recall in LLM contexts
